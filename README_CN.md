# BPMN Editor

这是我在使用 [bpmn-js][16] 框架的时候整理的使用方法，更全更详细的使用方法请参考[官方例子][15]

- [basic][1] : 基础用法  [演示地址][8]
- [custom-menu][2] : 自定义菜单项  [演示地址][9]
- [custom-palette][3] : 自定义左侧工具栏  [演示地址][10]
- [custom-property][4] : 自定义属性栏  [演示地址][11]
- [custom-toolbar][5] ： 自定义顶部导航栏  [演示地址][12]
- [i18n][6] : 国际化  [演示地址][13]
- [i18n][7] : 键盘快捷键  [演示地址][14]
